Dual-boot_avec Debian 13_et grub

J'ai essayé d'installer tiny core linux x86 avec une carte SD sur un ASUS EEE 1001PX, mais ça ne bootait pas. J'ai installé Debian 13 ensuite et j'ai modifié le grub de cette manière.

Pour commencer, à partir du menu grub, je suis passé dans la ligne de commande et j'ai tapé

set root=(hd1,1) linux /boot/vmlinuz initrd /boot/core.gz boot

Ensuite, j'ai créé un /etc/grub.d/40_custom et j'ai ajouté un menuentry

menuentry "tiny-core-linux-16.1" { set root=(hd1,1) linux /boot/vmlinuz initrd /boot/core.gz }
j'ai aussi modifié /etc/default/grub pour GRUB_DEFAULT=2 et ensuite j'ai fait sudo update-grub. Après redémarrage, je bootais de la carte SD avec tiny core par défaut. Je me suis arrêté là.
