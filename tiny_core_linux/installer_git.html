Installer git

Cette page fait suite à Du début jusqu'à tmux .

On peut installer git avec tce-load -wi git.tcz. On peut s'assurer que l'extension a bien été installée en faisant 

cd /mnt/sda2/tce
cat onboot.lst
ls optional/
On devrait y voir git.tcz. Notez que tce est dans sda2 pour moi. Selon l'ordinateur, il faudra adapter à la bonne partition.
